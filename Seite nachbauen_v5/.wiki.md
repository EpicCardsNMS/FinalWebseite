# Project Summary
EpicCards is a web application for buying and selling collectible trading cards, including popular games like Pokémon and Magic: The Gathering. The platform enhances user engagement by simplifying the selling process and providing quick, non-binding purchase offers through an intuitive interface.

# Project Module Description
- **Backend**: Manages request processing, authentication, and email notifications for inquiries.
- **Frontend**: Provides a user-friendly interface for card selection and inquiry submissions, integrating a local email client for seamless communication.

# File Description Inventory
- **app/backend/main.py**: Entry point for the backend API.
- **app/backend/routers/inquiry.py**: Manages card inquiry submissions and email notifications.
- **app/frontend/src/components/CardSelector.tsx**: Updated to open the user's email client with a pre-filled template based on card selection, removing instructions for users.
- **app/frontend/src/components/Header.tsx**: Updated to increase the size of the EpicCards logo.
- **app/frontend/index.html**: Main HTML file for the frontend application.

# Technology Stack
- **Backend**: FastAPI, Python
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL (assumed based on common practices)
- **Email Service**: Local email client via `mailto:` links

# Usage
1. **Install Backend Dependencies**:
   ```bash
   pip install -r app/backend/requirements.txt
   ```
2. **Install Frontend Dependencies**:
   ```bash
   cd app/frontend
   pnpm install
   ```
3. **Run Backend**:
   ```bash
   uvicorn app.backend.main:app --reload
   ```
4. **Run Frontend**:
   ```bash
   pnpm run dev
   ```
